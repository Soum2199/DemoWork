### âœ… Definition Recap:

A **Strong Number** is a number in which the **sum of the factorials of its digits** equals the number itself.

**Example:**

* 145 is a Strong Number:
  ( 1! + 4! + 5! = 1 + 24 + 120 = 145 ) â†’ âœ… YES

### ðŸ”µ C

```c
#include <stdio.h>

int factorial(int n) {
    int fact = 1;
    for(int i = 1; i <= n; i++)
        fact *= i;
    return fact;
}

int main() {
    int num, sum = 0, temp, digit;
    scanf("%d", &num);
    temp = num;

    while (temp > 0) {
        digit = temp % 10;
        sum += factorial(digit);
        temp /= 10;
    }

    if (sum == num)
        printf("YES\n");
    else
        printf("NO\n");

    return 0;
}
```
