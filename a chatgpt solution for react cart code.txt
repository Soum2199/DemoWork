### üîë Problems in your code:

1. `const [cart, setCart] = useState('')`

   * `cart` should be an **array**, not a string, because you want to store multiple products.

   ‚úÖ Use:

   ```js
   const [cart, setCart] = useState([]);
   ```

2. In `carting()`, you‚Äôre doing:

   ```js
   setCart(cart)
   ```

   * This doesn‚Äôt change anything ‚Äî it just sets `cart` to itself.
     You should **add a new product** into the `cart`.

   ‚úÖ Use:

   ```js
   function carting(product) {
     setCart([...cart, product]); // push product into array immutably
   }
   ```

3. In `Produce`, you‚Äôre not sending the product to `carting`.

   * You should call `carting(i)` when "Add to cart" is clicked.

4. In `Cart`, you‚Äôre not displaying cart items. Just `"Cart"`.

#### `Cart.js`

```jsx
import React from 'react';

function Cart({ cart }) {
  return (
    <div>
      <h2>Cart</h2>
      {cart.length === 0 ? (
        <p>No items in cart.</p>
      ) : (
        <ul>
          {cart.map((item, idx) => (
            <li key={idx}>
              {item.names} - Rs.{item.price}
            </li>
          ))}
        </ul>
      )}
    </div>
  );
}

export default Cart;
```
