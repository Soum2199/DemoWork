## üß© 1. List-Specific (Public) Methods

These are the everyday methods you‚Äôll actually use:

| Method                             | Description                                                           |
| :--------------------------------- | :-------------------------------------------------------------------- |
| `append(x)`                        | Add an item `x` to the end of the list.                               |
| `extend(iterable)`                 | Add all items from another iterable (list, tuple, etc.) to the end.   |
| `insert(i, x)`                     | Insert item `x` at position `i`.                                      |
| `remove(x)`                        | Remove the first occurrence of `x`. Raises `ValueError` if not found. |
| `pop([i])`                         | Remove and return item at index `i` (default: last item).             |
| `clear()`                          | Remove all items (empty the list).                                    |
| `index(x[, start[, end]])`         | Return the index of the first occurrence of `x`.                      |
| `count(x)`                         | Return how many times `x` appears.                                    |
| `sort(*, key=None, reverse=False)` | Sort the list in place.                                               |
| `reverse()`                        | Reverse the list in place.                                            |
| `copy()`                           | Return a shallow copy of the list.                                    |

---

## ‚öôÔ∏è 2. Special (Dunder) Methods

These are used internally or for operator overloading.

| Method                                                     | Description                                                    |
| :--------------------------------------------------------- | :------------------------------------------------------------- |
| `__add__(self, other)`                                     | Enables `+` operator to concatenate lists.                     |
| `__iadd__(self, other)`                                    | Enables `+=` operator for extending lists.                     |
| `__mul__(self, n)`                                         | Enables `*` operator for repetition.                           |
| `__rmul__(self, n)`                                        | Enables right-side repetition (`n * list`).                    |
| `__imul__(self, n)`                                        | Enables in-place multiplication (`list *= n`).                 |
| `__getitem__(self, index)`                                 | Access item(s) via indexing or slicing.                        |
| `__setitem__(self, index, value)`                          | Assign to item(s) via indexing or slicing.                     |
| `__delitem__(self, index)`                                 | Delete item(s) via indexing or slicing.                        |
| `__contains__(self, item)`                                 | Enables `in` operator.                                         |
| `__len__(self)`                                            | Enables `len(list)`.                                           |
| `__iter__(self)`                                           | Returns an iterator for looping (`for x in list`).             |
| `__reversed__(self)`                                       | Called by `reversed(list)`.                                    |
| `__eq__`, `__ne__`, `__lt__`, `__le__`, `__gt__`, `__ge__` | Comparison operators.                                          |
| `__repr__(self)`                                           | Official string representation, e.g. `[1, 2, 3]`.              |
| `__str__(self)`                                            | Informal string representation (same as `__repr__` for lists). |
| `__hash__(self)`                                           | Lists are *not hashable* ‚Äî this is disabled.                   |
| `__sizeof__(self)`                                         | Returns memory size of the list object.                        |

---

## üß† 3. Inherited from `object`

Every Python class inherits these:

| Method / Attribute                 | Description                                         |
| :--------------------------------- | :-------------------------------------------------- |
| `__class__`                        | The class the object belongs to (`<class 'list'>`). |
| `__init__(self, iterable=())`      | Constructor; initializes list from iterable.        |
| `__new__(cls, *args, **kwargs)`    | Creates a new instance (before `__init__`).         |
| `__getattribute__(self, name)`     | Called when an attribute is accessed.               |
| `__setattr__(self, name, value)`   | Called when setting an attribute.                   |
| `__delattr__(self, name)`          | Called when deleting an attribute.                  |
| `__dir__(self)`                    | Returns list of valid attributes for `dir(obj)`.    |
| `__doc__`                          | The docstring of the class.                         |
| `__format__(self, format_spec)`    | Called by `format(obj, spec)`.                      |
| `__reduce__`, `__reduce_ex__`      | Used by `pickle` for object serialization.          |
| `__init_subclass__(cls, **kwargs)` | Called when subclassing `list`.                     |
| `__subclasshook__(cls, subclass)`  | Determines if a class is considered a subclass.     |
